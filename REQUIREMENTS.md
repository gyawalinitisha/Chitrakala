# Project Requirements

To run this project, you need the following software installed on your machine:

## 1. Node.js
- **Required Version**: v16.0.0 or higher
- **Purpose**: Runs the backend server and the frontend build tools (Vite).
- **Download**: [nodejs.org](https://nodejs.org/)

## 2. MongoDB
- **Required Version**: v5.0 or higher
- **Purpose**: Database to store user accounts, artwork data, and orders.
- **Download**: [MongoDB Community Server](https://www.mongodb.com/try/download/community)
- **Installation Note**: Ensure MongoDB is running as a service (usually on port 27017).

## 3. Git
- **Purpose**: For version control.
- **Download**: [git-scm.com](https://git-scm.com/)

---

## Quick Start
1.  **Install Dependencies**:
    - Backend: `cd server && npm install`
    - Frontend: `npm install`
2.  **Start Database**: Ensure MongoDB service is running.
3.  **Start Backend**: `cd server && npm run dev`
4.  **Start Frontend**: `npm run dev` in a separate terminal.
